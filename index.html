Privacy Policy – MedTracker

Last updated: 2026

1. Data Collection

MedTracker allows users to manually enter and store medication-related data.

  • Medication names
  • Dosage information
  • Intake times
  • Daily medication logs (taken / missed)

2. Data Storage

All data is stored locally on your device.

3. Data Usage

Used only inside the app.

4. Data Sharing

No data sharing.

5. Data Security

Device security responsibility.

6. Data Retention & Deletion

Stored until deleted.

  • Delete entries in app
  • Uninstall deletes all data

No remote access.

7. Permissions

No sensitive permissions.

8. Changes

Updates possible.
